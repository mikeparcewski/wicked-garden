Validates auto-advance projection for a minimal-tier project (complexity_score=1.5, below the threshold of 3). After `wicked.gate.decided` returns APPROVE, the bus consumer fires `wicked.phase.auto_advanced` — an audit-only event that produces the same phase/project state update as a normal `wicked.phase.transitioned`. The expected projection is identical to the `single_phase_approve` shape, confirming that `wicked.phase.auto_advanced` is handled by the same projection rule as `wicked.phase.transitioned`. Note that `gate_score` and `gate_verdict` on the phase row are set by the preceding `wicked.gate.decided`, not by the auto-advance event itself.
