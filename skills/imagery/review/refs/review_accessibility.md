# Visual Review: Accessibility (a11y)

This reference focuses on making visual assets accessible to all users, including those with visual impairments.

## Core Objectives
- Check color contrast ratios for readability.
- Ensure information is not conveyed by color alone.
- Verify visual clarity and lack of "visual noise" that could be distracting.

## Review Workflow
1.  **Contrast Analysis:** Use tools or multimodal analysis to check contrast between text/icons and their backgrounds.
2.  **Color-Blind Simulation:** Analyze how the image appears in grayscale or simulated color-blindness modes.
3.  **Readability Check:** Ensure any text within the image is legible at its intended display size.

## Review Criteria
- Does the contrast meet WCAG AA or AAA standards?
- Can a user distinguish between elements without relying on color?
- Is the focal point clear and undistorted?
